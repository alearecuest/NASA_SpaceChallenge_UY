Sobre el Proyecto

Este proyecto es una aplicación interactiva diseñada para rastrear, visualizar y simular el impacto de objetos cercanos a la Tierra (NEOs), como asteroides y cometas. Utilizamos datos en tiempo real de las APIs de la NASA para proporcionar una herramienta educativa y de concienciación sobre la defensa planetaria.

Recursos y Tecnologías

Nuestro proyecto se basa en datos fiables y herramientas de vanguardia proporcionadas por la NASA y otras organizaciones.

  • API Principal: JPL SSD & CNEOS API
  • Datos de Acercamiento: Close Approach Data API
  • Datos de Bólidos: Fireball API
  • Efemérides: Horizons API
